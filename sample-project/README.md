# Golden Fork Restaurant Project

## Project Overview
Golden Fork Restaurant is a multi-page HTML-only website project designed for a restaurant business. It includes important restaurant pages such as home, menu, about, services, reservations, chefs, events, offers, testimonials, FAQ, gallery, and contact.

## How to Collaborate with Git and GitHub

### 1. Clone the repository
```bash
git clone <repository-link>
````

### 2. Enter the project folder

```bash
cd restaurant-website
```

### 3. Create your branch

```bash
git checkout -b your-name-branch
```

Example:

```bash
git checkout -b daniel-branch
```

### 4. Work on your assigned files

Edit only your pages first.

### 5. Check changes

```bash
git status
```

### 6. Add files

```bash
git add .
```

### 7. Commit changes

```bash
git commit -m "Create homepage and navigation"
```

### 8. Push your branch

```bash
git push -u origin your-name-branch
```

### 9. Open a Pull Request on GitHub

After pushing, go to GitHub and create a Pull Request to merge your branch into `main`.

### 10. Pull latest changes before starting new work

```bash
git checkout main
git pull origin main
git checkout your-name-branch
git merge main
```

## Branching Rules

* Never work directly on `main`
* Always create your own branch
* Pull the latest changes before starting new work
* Use clear commit messages
* Test links between pages before pushing

## Final Notes

This project is suitable for beginners learning HTML structure, website navigation, teamwork, and Git collaboration.
